| 📝 **Resumo - Resume Optimization Platfom**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| A full-stack resume-to-job-description matcher with a hybrid ATS scoring engine: a spaCy PhraseMatcher extracts skills against a canonical dictionary with alias overrides, then rapidfuzz catches near-variants like Node.js vs NodeJS that exact matching misses. A weighted 50/30/20 blend of keyword overlap, structural ATS checks (contact info, section headers, garbled-text detection), and completeness signals produces a single score with a full breakdown. FastAPI backend handles pytesseract OCR for job description screenshots, pdfplumber/python-docx resume parsing, and Groq-generated suggestions constrained by a system prompt to only rephrase content already in the resume, never invent unearned skills or metrics, with schema validation and a retry on malformed output. Multi-section PDF reports are built in-memory with fpdf2 and streamed back statelessly, behind a Vite + React + Tailwind CSS frontend built around a three-step paste-upload-analyze flow. |
